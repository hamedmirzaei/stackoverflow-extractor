        This demo is provided using Java programming language and is based on Spring Boot framework.
        The template engine for generating web pages is Thymeleaf.

        As the task description dictates, you can see too menus Newest and Most Voted which their goal
        is to show top 10 newest and top 10 most voted Android questions from StackOverflow website respectively.

        Our demo uses JSoup -which is a powerful Java library- to extract information from
        pages of stackoverflow.com website
